
# Hello Bundle Starpi CMS & Next jS Landing Page


## Tech Stack

**Client:** NextJS 15 , TailwindCSS 4 , React 19

**Server:** Strapi 5, Postgres 16


## Dependencies

**postgres** 16.6

**node** v23.9.0

## Setup guide

### Strapi Server

Path
```bash
    cd api
```

Install packages
```bash
    npm install
```

Run application
```bash
    npm run dev
```

### client

Path
```bash
    cd client
```

Install packages
```bash
    npm install
```

Run application
```bash
    npm run dev
```
